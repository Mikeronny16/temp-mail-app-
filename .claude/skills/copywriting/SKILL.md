# Skill: copywriting

Viral conversion copy — SE Asia Gen Z audience, $2–$10 digital products.

## Formulas

### Hero Headline
```
[Bold claim] + [who it's for] + [without the pain]
"Generate 10x better proposals — in 30 seconds — without staring at a blank page"
```

### CTA Buttons
- NOT: "Submit" / "Click here"  
- YES: "Get My Copy Now" / "Start Free" / "Try It — It's Free" / "Unlock Access"

### Social Proof Line
```
"Used by 1,200+ freelancers in Myanmar & Southeast Asia"
"4.9★ from 340 users this week"
```

### Price Anchoring
```
❌ $39/month (too expensive framing)
✅ Less than a cup of coffee — $5 one-time
✅ Save 3 hours per proposal — for $5
```

### Urgency (ethical)
```
"Only 47 spots left at this price"
"Price increases after 100 users"
```

## Myanmar-specific
- Use "ကိုယ့်ရဲ့" ownership framing
- Highlight KBZPay / Wave — familiar trust signal
- "Myanmar freelancers" — hyper-local beats global
- Mobile screenshot in hero — 80% browse on phone

## Email Subject Lines (for manual payment)
```
"Your [Product] access is ready 🎉"
"Payment confirmed — here's your download link"
"[Name], your credits are live!"
```

## Rules
- Max 1 exclamation mark per section
- Numbers beat adjectives: "10x faster" > "much faster"
- Burmese copy: shorter sentences, direct, no formal ထောင် tone
- Every section answers: "What's in it for ME?"
